Spatial/AR Computing

Pure-C++ 3D math and perspective projection for terminal UIs. Position widgets in 3D space, render them depth-sorted onto a BrailleCanvas, and add a HUD overlay — no GPU required.

#include "ftxui/ui/spatial.hpp"
📐

Vec3 & Mat4

3D vector math and column-major 4×4 matrices: translate, scale, rotate, perspective, lookAt.

📷

Camera3D

Perspective camera with orbit, zoom, and pan. View + projection matrices built automatically.

🧩

SpatialScene

Depth-sorted widget rendering with content blitting, bounding boxes, and optional auto-orbit.

🗺️

SpatialLayouts

Arrange widgets in Circle, Grid, Sphere, Stack, or Fan formations automatically.

🎯

HUD

Head-up display overlay: corner elements, crosshair, and compass rendered over the 3D scene.

🚀

SpatialDock

One-line helper: wrap any components in an auto-orbiting 3D dock.

Quick Start

#include "ftxui/ui/spatial.hpp"
#include "ftxui/ui/app.hpp"
#include "ftxui/component/component.hpp"
using namespace ftxui;
using namespace ftxui::ui;

int main() {
  // 1. Create a scene with a perspective camera
  SpatialScene scene(Camera3D{{0, 2, 8}});

  // 2. Add widgets at 3D positions
  for (int i = 0; i < 6; i++) {
    SpatialWidget w;
    w.content = Renderer([] { return text("Hello") | border; });
    w.content_width = 10;
    w.content_height = 3;
    scene.AddWidget(std::move(w));
  }

  // 3. Arrange in a circle and start auto-orbit
  SpatialLayouts::Circle(scene, 4.0f);
  scene.StartOrbit(20.0f);

  // 4. Build interactive component and run
  auto comp = scene.BuildComponent();
  ui::App app;
  app.Add(comp);
  app.Run();
}

API Reference

Vec3 — 3D Vector

Simple 3D float vector with math operations.

struct Vec3 {
  float x, y, z;
  Vec3 operator+(Vec3) const;    // addition
  Vec3 operator-(Vec3) const;    // subtraction
  Vec3 operator*(float) const;   // scalar multiply
  Vec3 cross(Vec3) const;        // cross product
  float dot(Vec3) const;         // dot product
  float length() const;          // euclidean length
  Vec3 normalized() const;       // unit vector
  static Vec3 Zero();
  static Vec3 Up();      // {0,1,0}
  static Vec3 Forward(); // {0,0,-1}
};

Mat4 — 4×4 Matrix

Column-major 4×4 transformation matrix (OpenGL convention). Index = col×4 + row.

struct Mat4 {
  float m[16];  // column-major storage
  static Mat4 Identity();
  static Mat4 Translation(Vec3 t);
  static Mat4 Scale(Vec3 s);
  static Mat4 RotationX(float radians);
  static Mat4 RotationY(float radians);
  static Mat4 RotationZ(float radians);
  static Mat4 Perspective(float fov_y, float aspect,
                          float near, float far);
  static Mat4 LookAt(Vec3 eye, Vec3 target, Vec3 up);
  Mat4 operator*(const Mat4&) const;
  Vec3 Transform(Vec3 v) const; // perspective divide
};

Camera3D

Perspective camera. Builds view + projection matrices and supports interactive controls.

struct Camera3D {
  Vec3 position = {0, 0, 5};
  Vec3 target   = {0, 0, 0};
  Vec3 up       = {0, 1, 0};
  float fov     = 60.0f;        // degrees
  float near_plane = 0.1f;
  float far_plane  = 1000.0f;

  Mat4 ViewMatrix() const;
  Mat4 ProjectionMatrix(float aspect) const;
  Mat4 ViewProjection(float aspect) const;

  void OrbitAround(Vec3 center, float delta_yaw, float delta_pitch);
  void Zoom(float delta);       // move along view direction
  void Pan(Vec3 delta);         // translate camera + target
};

SpatialWidget

A UI component placed in 3D space. Set content to any FTXUI Component for full rendering.

struct SpatialWidget {
  int id;                        // assigned by scene
  Vec3 position;                 // world-space position
  Vec3 rotation;                 // Euler angles (radians)
  Vec3 scale  = {1, 1, 1};
  float opacity = 1.0f;

  ftxui::Component content;     // rendered and blitted
  int content_width  = 20;      // in terminal columns
  int content_height = 10;      // in terminal rows
  bool visible = true;
  std::string label;             // shown when no content
};

SpatialScene

Manages a collection of SpatialWidgets, renders them depth-sorted to a BrailleCanvas.

SpatialScene scene(Camera3D{{0, 2, 8}});

int id = scene.AddWidget(widget);
scene.RemoveWidget(id);
SpatialWidget* w = scene.GetWidget(id);
scene.UpdateWidget(id, [](SpatialWidget& w) {
  w.position.x += 1.0f;
});

scene.SetCamera(cam);
Camera3D& cam = scene.camera();

// Render to braille canvas element (canvas_cols × canvas_rows chars)
ftxui::Element el = scene.Render(80, 24);

// Project a world point to screen coordinates
auto proj = scene.Project({0,0,0}, 80, 24);
// proj.x, proj.y  — screen position in characters
// proj.depth      — distance in front of camera
// proj.visible    — false if behind camera

// Build interactive component with mouse orbit/zoom
ftxui::Component comp = scene.BuildComponent();

// Background auto-orbit
scene.StartOrbit(30.0f);  // 30 degrees/second
scene.StopOrbit();

SpatialLayouts

Automatically arrange all widgets in a scene.

// Evenly spaced around a circle
SpatialLayouts::Circle(scene, /*radius=*/3.0f, /*y=*/0.0f);

// 3×3 (or N×cols) grid on XZ plane
SpatialLayouts::Grid(scene, /*cols=*/3, /*spacing=*/2.5f);

// Golden-angle spiral on a sphere surface
SpatialLayouts::Sphere(scene, /*radius=*/4.0f);

// Stack vertically along Y axis
SpatialLayouts::Stack(scene, /*y_step=*/1.5f);

// Fan arc facing the camera
SpatialLayouts::Fan(scene, /*arc_degrees=*/120.0f, /*distance=*/3.0f);

HUD

Overlay elements on top of a spatial scene element using dbox.

HUD hud;
// corner: 0=top-left, 1=top-right, 2=bottom-left, 3=bottom-right
hud.AddCorner(ftxui::text("FPS: 60"), 1);
hud.AddCrosshair(true);   // center + symbol
hud.AddCompass(true);     // N/E/S/W directions

ftxui::Element scene_el = scene.Render(80, 24);
ftxui::Element final    = hud.Render(std::move(scene_el));

SpatialDock & FloatingWindow

Convenience constructors for common patterns.

// Wrap multiple components in an auto-orbiting circle
ftxui::Component dock = SpatialDock({comp_a, comp_b, comp_c});

// Single component floating at a 3D position
ftxui::Component win = FloatingWindow(
    my_component, {2.0f, 0.0f, -1.0f}, "My Window");
