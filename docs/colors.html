🎨 Colors & Rich Text

Full terminal colorization system: TextStyle fluent builder, RichText markup, AnsiParser, 256-color palettes, TrueColor gradients, and convenience helpers.

Quick Start

#include "ftxui/ui/rich_text.hpp"
using namespace ftxui;
using namespace ftxui::ui;

// Fluent style builder
auto el1 = TextStyle{}.Bold().Red()("Hello, world!");

// Rich markup string
auto el2 = RichText::Element("[bold][cyan]FTXUI[/] Rich Text[/bold] is [green]great[/green]!");

// ANSI passthrough
auto el3 = AnsiParser::Element("\033[1;32mGreen Bold\033[0m via ANSI");

// Convenience helpers
auto el4 = RgbText("Orange", 255, 128, 0);
auto el5 = ColorGradient(Color::RGB(255,0,0), Color::RGB(0,0,255), 40);

TextStyle — Fluent Builder

TextStyle is a zero-overhead fluent builder that accumulates all terminal attributes and applies them to any ftxui::Element.

Colors

MethodDescription
.Fg(Color)Foreground color
.Bg(Color)Background color
.Fg(r,g,b)TrueColor foreground
.Bg(r,g,b)TrueColor background
.Fg256(n)256-palette foreground
.Bg256(n)256-palette background
.Red() .Green() .Blue()Named shortcuts
.BrightRed() .BrightBlue()…Bright variants

Attributes

MethodEffect
.Bold()Bold text
.Dim()Dimmed / faint
.Italic()Italic
.Underline()Underlined
.Blink()Blinking
.RapidBlink()Rapid blink (→ blink)
.Reverse()Swap fg/bg (inverted)
.Hidden()Hidden / invisible
.Strikethrough()Strike-through

Applying Styles

TextStyle s;
s.Bold().Fg(255, 165, 0).Underline();   // chain methods

// Option A – call operator
auto el = s("Orange bold underlined text");

// Option B – apply to existing element
auto el2 = s.Apply(ftxui::text("Same result"));

// Option C – pipe operator (via decorator)
auto el3 = ftxui::text("text") | ftxui::bold | ftxui::color(Color::Red);

RichText — Inline Markup

Parse and render bracket-based inline markup tags. Tags stack and can be closed individually ([/tag]) or all at once ([/]).

Supported Tags

TagEffect
[bold]Bold text
[dim]Dim text
[italic]Italic
[underline]Underlined
[blink]Blinking
[reverse]Inverted colors
[strikethrough]Strike-through
TagEffect
[red] … [white]Named fg colors
[bright_red] etc.Bright variants
[bg_red] etc.Background colors
[rgb(r,g,b)]TrueColor fg
[bg_rgb(r,g,b)]TrueColor bg
[/] or [/tag]Close tag(s)

Example

auto el = RichText::Element(
    "[bold][yellow]Warning:[/] "
    "[red]File not found[/red] — "
    "[dim]check your path[/dim]"
);

API

// Parse to spans (for custom rendering)
std::vector<Span> spans = RichText::Parse(markup);

// Render directly to Element
ftxui::Element el = RichText::Element(markup);

// Render pre-parsed spans
ftxui::Element el2 = RichText::Render(spans);

// Wrap as a Component
ftxui::Component c = RichText::Component(markup);

AnsiParser — ANSI Escape Sequences

Converts raw ANSI SGR escape sequences from any source (shell command output, log files, etc.) into styled FTXUI elements.

Supported SGR Codes

Code(s)Effect
0Reset all attributes
1–9Bold, Dim, Italic, Underline, Blink, Rapid Blink, Reverse, Hidden, Strikethrough
30–37Standard foreground colors (black–white)
38;2;r;g;bTrueColor foreground
38;5;n256-palette foreground
40–47Standard background colors
48;2;r;g;bTrueColor background
48;5;n256-palette background
90–97Bright foreground colors
100–107Bright background colors

Example

// From a shell command that emits ANSI output
std::string output = RunCommand("ls --color=always");
ftxui::Element el = AnsiParser::Element(output);

// Strip ANSI for plain text
std::string plain = AnsiParser::Strip(output);

// Parse to spans for custom processing
auto spans = AnsiParser::Parse(output);

Integration

// Include everything at once
#include "ftxui/ui.hpp"

// Or just the rich text module
#include "ftxui/ui/rich_text.hpp"

Link against ftxui::ui (which transitively includes ftxui::component, ftxui::dom, and ftxui::screen).

target_link_libraries(my_app PRIVATE ui)
