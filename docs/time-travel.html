Quick Start API Reference

Quick Start

1. Track reactive values

#include "ftxui/ui/time_travel.hpp"

// Track any Reactive<T> with a single macro:
auto counter = std::make_shared<Reactive<int>>(0);
BADWOLF_TRACK(counter, "counter");

// Or with a custom serializer:
StateRecorder::Global().Track<int>(counter, "counter",
    [](const int& v) { return JsonValue(v); });

2. Wrap your app вЂ” press F12 to open the debugger

auto app_ui = Renderer(container, [&]() { /* your UI */ });

// Wrap with time-travel panel (toggled by F12):
auto root = WithTimeTravel(app_ui);

auto screen = ScreenInteractive::Fullscreen();
screen.Loop(root);

3. Or embed the debugger directly

auto recorder = std::make_shared<StateRecorder>();
auto debugger = TimeTravelDebugger(recorder);

// Use arrow keys / space to navigate:
//   в†ђ в†’    step backward / forward
//   Home   go to first snapshot
//   End    go to last snapshot
//   Space  play / pause
auto layout = Container::Vertical({app_ui, debugger});

Key Concepts

рџ“ј

StateRecorder

Ring-buffer-backed recorder that stores up to N snapshots. Each snapshot captures before / after JSON values, a label, a key, a trigger string, and a high-resolution timestamp.

вЏЇ

TimelinePlayer

Navigates snapshots by index or time. Supports step-by-step navigation and background-thread playback at configurable speed (0.25Г— вЂ“ 4Г—). Fires position-change callbacks on every move.

рџ”Ќ

TimeTravelDebugger

A full FTXUI Component showing a scrubber bar, timeline list, before/after diff panel, and playback controls. Embed directly in your layout or use WithTimeTravel for a toggleable overlay.

рџЏ·пёЏ

BADWOLF_TRACK

Convenience macro that registers a Reactive<T> with the global StateRecorder. Uses std::ostringstream by default; supply a custom serializer for richer JSON output.

API Reference

StateSnapshot

struct StateSnapshot {
  uint64_t id;
  std::chrono::steady_clock::time_point timestamp;
  std::string label;       // human-readable description
  std::string state_key;   // reactive variable name
  JsonValue before;        // value before change
  JsonValue after;         // value after change
  std::string trigger;     // e.g. "user_event", "unknown"
};

StateRecorder

// Record a change manually:
void Record(key, before, after, label = "", trigger = "unknown");

// Auto-record via Reactive<T>:
template<typename T>
void Track(shared_ptr<Reactive<T>>, key, serializer = nullptr);

// Query:
const vector<StateSnapshot>& Snapshots() const;
size_t Count() const;
void Clear();

// Persistence:
JsonValue ExportJson() const;
bool SaveToFile(const string& path) const;
static StateRecorder LoadFromFile(const string& path);

// Singleton:
static StateRecorder& Global();

TimelinePlayer

explicit TimelinePlayer(const StateRecorder&);

// Navigation:
void GoToSnapshot(size_t index);
void GoToTime(steady_clock::time_point t);
void StepForward();   void StepBackward();
void GoToStart();     void GoToEnd();

// Playback:
void PlayFrom(size_t index, double speed = 1.0);
void Pause();

// State:
size_t CurrentIndex() const;
const StateSnapshot& CurrentSnapshot() const;
bool AtStart() const;  bool AtEnd() const;  bool IsPlaying() const;

// Callbacks:
int  OnPositionChange(function<void(size_t, const StateSnapshot&)>);
void RemoveCallback(int id);

Free Functions

// Full interactive debugger component:
ftxui::Component TimeTravelDebugger(

    ftxui::Component app_component = nullptr);

// Wrap any component with F12-toggled debugger overlay:
ftxui::Component WithTimeTravel(
    ftxui::Component app,
    shared_ptr<StateRecorder> recorder = nullptr);

// Macro shorthand for Global() recorder:
BADWOLF_TRACK(reactive_ptr, "key");
