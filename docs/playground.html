⌨️ Keyboard Shortcuts

  • Tab Move focus to next widget
  • Shift+Tab Move focus to previous widget
  • ↑ ↓ ← → Navigate within components
  • Enter Activate focused button
  • Ctrl+C Exit (native); noop in browser

📦 How to Embed

  • 1 Build with emcmake cmake -DFTXUI_BUILD_WASM=ON
  • 2 Copy badwolf_playground.js, .wasm and wasm_shell.html to your web server.
  • 3 Serve with COOP/COEP headers — required for SharedArrayBuffer (used by pthreads).
  • 4 Embed with <iframe src="wasm_shell.html">.
