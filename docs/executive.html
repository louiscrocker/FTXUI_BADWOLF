The Problem

The Terminal Renaissance Has a Tooling Gap

Cloud-native infrastructure runs in terminals. The tools to build for it haven't kept up.

📈

Terminal is Back

Kubernetes, Docker, cloud-native tooling — all terminal-first. 87% of DevOps engineers spend >4 hours/day in a terminal. Remote-first work doubled terminal UI usage since 2020.

🦕

Existing Options Are Ancient

ncurses is 1980s C. Textual requires Python & 50MB RAM. Bubbletea/Ratatui require language switches. None offer charts, maps, or reactive state out of the box.

🎯

C++ Developers Underserved

4.4M active C++ developers — in HFT, embedded, defense, HPC — have NO good TUI option. They either write ncurses spaghetti or skip the UI entirely.

Use Cases

What Teams Are Building with FTXUI

🖥️

System Monitoring

ftop — real-time CPU, memory, disk I/O, and network metrics. Interactive process browser with kill, renice, and strace. Used by SREs at hyperscale companies to replace htop on production fleets.

🌍

Operations Centers

Military and defense C2 systems. World map with threat overlays, missile trajectory arcs, unit positions. The GeoJSON map + globe are purpose-built for real-time situational awareness dashboards.

💹

Trading Dashboards

Sub-millisecond render keeps HFT firms competitive. P&L displays, order book depth charts, position monitors, and risk dashboards — all in C++ with no runtime overhead from a UI framework.

🔧

DevOps Tooling

Kubernetes status dashboards, deployment pipelines with live log streaming, incident response tools. The ProcessPanel + LogPanel + GeoMap combination maps perfectly to cluster operations.

🗄️

Database Management

ftdb — SQLite and PostgreSQL browser with query runner, schema explorer, and live result tables. Uses SortableTable with pagination for millions of rows.

📊

Scientific Computing

HPC cluster dashboards showing job status, node utilization sparklines, and simulation progress. BrailleCanvas renders custom scientific visualizations at terminal resolution.

🚀

Embedded / Edge Systems

IoT device status panels, industrial control interfaces, edge computing dashboards. Zero dependencies and ~2MB footprint make FTXUI viable on constrained systems where Python is impossible.

🔐

Security & Intelligence

Network scanners, threat intelligence dashboards, log analyzers. Air-gapped environments where web UIs are forbidden and Python runtimes aren't installed.

🎮

Game Server Admin

Real-time player counts, server status grids, world map visualizations of player locations, and live event streams. The reactive state system is ideal for game telemetry.

📡

Telecommunications

Network topology visualization, call flow diagrams, fault management dashboards. The GeoMap with arc overlays perfectly represents network links and their geographic routing.
