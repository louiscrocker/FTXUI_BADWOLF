JSON as a First-Class Data Type

JsonValue, a handwritten RFC 8259 parser, JSONPath traversal, reactive JSON state, and ready-made tree/table/form UI components — all zero external dependencies.

Quick Start API Reference

Quick Start

1. Parse & Query

#include "ftxui/ui/json.hpp"
using namespace ftxui::ui;

// Parse
auto v = Json::Parse(R"({"name":"Alice","scores":[95,87,100]})");

// Access
std::string name = v["name"].as_string();  // "Alice"
int first_score  = v["scores"][0u].as_int(); // 95

// Check presence
if (v.has("name")) { /* ... */ }

// Safe access (no exception on mismatch)
int age = v["age"].get_int(0);  // 0 — key absent

2. Build & Serialize

auto obj = JsonValue::Object();
obj.set("host", JsonValue{"web-01"});
obj.set("cpu",  JsonValue{12.5});

auto arr = JsonValue::Array();
arr.push(obj);

// Compact
std::string s = Json::StringifyCompact(arr);
// → [{"cpu":12.5,"host":"web-01"}]

// Pretty
std::string p = Json::StringifyPretty(arr, 2);

3. JSONPath

auto root = Json::Parse(R"({"users":[{"name":"Alice"},{"name":"Bob"}]})");

JsonPath p("$.users[0].name");
auto val = p.Get(root);          // "Alice"

// Wildcard — collect all matching nodes
JsonPath all("$.users[*].name");
auto names = all.GetAll(root);   // ["Alice", "Bob"]

// Mutate by path
JsonPath("$.users[0].name").Set(root, JsonValue{"Eve"});

4. ReactiveJson

auto state = std::make_shared<ReactiveJson>(
    Json::Parse(R"({"count":0})"));

// Subscribe
state->OnChange([](const JsonValue& v) {
    // called on every mutation
});

// Mutate by path — triggers callbacks + UI redraw
state->Set("$.count", JsonValue{42});

// Patch (merge)
state->Patch(Json::Parse(R"({"extra":"data"})"));

5. UI Components

// Syntax-highlighted element
auto elem = JsonElement(v);

// Interactive collapsible tree
auto tree = JsonTreeView(v, "root");

// Auto-columns table from JSON array
auto table = JsonTableView(json_array);

// Auto-generated form from JSON object schema
auto form = JsonForm(schema, [](JsonValue submitted) {
    // handle submitted data
});

// Live diff view
auto diff = JsonDiff(version_a, version_b);

API Reference

JsonValue

Self-contained JSON value: null, bool, number, string, array, or object. Value semantics — copy is always a deep clone.

Member Description
JsonValue()Null value
JsonValue(bool/int/double/string)Typed constructors
JsonValue::Object() / Array()Empty container factories
is_null/bool/number/string/array/object()Type predicates
as_bool/number/int/string/array/object()Typed access — throws on mismatch
get_bool/number/int/string(default)Safe access — returns default on mismatch
operator[](key/idx)Object/array subscript
has(key) / set(key, val) / erase(key)Object mutation
push(val) / pop() / size() / empty()Array mutation
operator== / !=Deep equality

Json:: namespace

Function Description
Parse(text)RFC 8259 parse — throws on error
ParseSafe(text, error*)Parse — returns null + sets error string
Stringify(val, indent)Serialize to JSON string
StringifyCompact(val)No whitespace
StringifyPretty(val, indent)Indented pretty-print
Merge(a, b)Shallow object merge (b overwrites a)
Clone(val)Deep clone

JsonPath

Subset of JSONPath: $.key, $.arr[0], $.arr[*], $.a.b.c.

Member Description
JsonPath(path_str)Parse path expression
Get(root)First matching value (null if not found)
Set(root, val)Mutate in-place
GetAll(root)All wildcard matches
valid()Path parsed successfully

ReactiveJson

Reactive wrapper around JsonValue — subscribers are notified and the UI refreshes on every mutation.

Member Description
Get()Read current value
Set(val)Replace entire value
Set(path, val)Mutate via JSONPath
Patch(partial)Shallow merge object patch
OnChange(cb)Subscribe to changes
AsReactive()Expose underlying Reactive<JsonValue>
