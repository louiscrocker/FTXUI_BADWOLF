Multiplayer Terminal Games

Real-time multiplayer games over TCP — up to 8 players, authoritative server model, braille canvas rendering, 30 fps game loop.

TCP Multiplayer 4 Built-in Games 30 FPS Loop 8 Players

Architecture

┌─────────────────────────────────────────────────────────────────────┐
│                    BadWolf Multiplayer Stack                         │
├──────────────────────────┬──────────────────────────────────────────┤
│  MultiplayerServer        │  MultiplayerClient                      │
│  ┌────────────────────┐   │  ┌────────────────────────────────────┐ │
│  │ AcceptLoop         │   │  │ Connect() → welcome/player_id      │ │
│  │ GameLoop (30fps)   │   │  │ RecvLoop() → GameState updates     │ │
│  │  ├ collect inputs  │   │  │ SendInput() → PlayerInput JSON     │ │
│  │  ├ OnTick callback │   │  │ OnStateUpdate callbacks            │ │
│  │  └ BroadcastState  │   └──────────────────────────────────────┘ │
│  │ ProcessClient(sock)│◄──────────────────────────────────────────  │
│  └────────────────────┘   Wire: [4-byte LE len][JSON payload]       │
│  PhysicsWorld (Vec2)      GameState: positions, scores, projectiles  │
└──────────────────────────┴──────────────────────────────────────────┘
                           │
            ┌──────────────▼──────────────┐
            │    MultiplayerRenderer       │
            │  GameState → BrailleCanvas   │
            │  Players: colored circles    │
            │  Projectiles: colored dots   │
            │  ScorePanel: hbox sidebar    │
            └──────────────────────────────┘

Quick Start

🖥️ Host a Game

./multiplayer_demo \
  --server \
  --port 29910 \
  --game shooter

Starts a TCP server on port 29910. Share your IP with friends.

🕹️ Join a Game

./multiplayer_demo \
  --host 192.168.1.10 \
  --port 29910 \
  --name "Alice"

Connect to a running server. Receive GameState at 30 fps.

⚙️ Embed in Your App

#include "ftxui/ui/multiplayer.hpp"
using namespace ftxui::ui;

// Server side
MultiplayerServer::Config cfg;
cfg.port = 29910;
auto server = std::make_shared<MultiplayerServer>(cfg);
server->OnTick([](GameState& gs, const std::vector<PlayerInput>& inputs) {
  gs.game_phase = "playing";
  for (auto& inp : inputs) {
    auto& pos = gs.player_positions[inp.player_id];
    if (inp.keys.left)  pos.x -= 1.0f;
    if (inp.keys.right) pos.x += 1.0f;
  }
});
server->Start();

// Client side
MultiplayerClient::Config ccfg;
ccfg.host = "localhost";  ccfg.port = 29910;
auto client = std::make_shared<MultiplayerClient>(ccfg);
client->Connect();
client->OnStateUpdate([](const GameState& gs) {
  // re-render with latest state
});

Wire Protocol

Uses the same framing as NetworkReactive and CollabSession for compatibility: 

[4-byte little-endian length][JSON payload bytes]

Server → Client (GameState, ~30/sec, compact JSON):
{"f":42,"ph":"playing","pp":{"0":{"x":10.5,"y":20.0},...},"ps":{"0":3},...}

Client → Server (PlayerInput):
{"frame":42,"keys":{"up":true,"down":false,...},"ax":0.5,"ay":-0.3}

Server → Client (Welcome, on connect):
{"type":"welcome","pid":0}

API Reference

GameState

struct GameState {
  uint64_t frame;
  std::map<int, Vec2>         player_positions;
  std::map<int, Vec2>         player_velocities;
  std::map<int, int>          player_scores;
  std::map<int, std::string>  player_names;
  std::map<int, Color>        player_colors;
  std::vector<tuple<Vec2,Vec2,Color>> projectiles;
  std::string game_phase;   // "waiting","playing","round_end","game_over"
  int round;
  float time_remaining;
  JsonValue ToJson() const;
  static GameState FromJson(const JsonValue&);
};

MultiplayerServer

auto server = std::make_shared<MultiplayerServer>(cfg);
server->OnTick([](GameState& gs, const std::vector<PlayerInput>& inputs) {
  // authoritative game logic here
});
server->Start();    // non-blocking
server->Stop();
server->PlayerCount();
server->CurrentState();

MultiplayerClient

auto client = std::make_shared<MultiplayerClient>(cfg);
bool ok = client->Connect();
int cbId = client->OnStateUpdate([](const GameState& gs) { /* render */ });
client->SendInput(inp);
client->RemoveCallback(cbId);
client->Disconnect();
